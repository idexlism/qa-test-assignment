# Баг-репорты 

## Задание 1: Баги по скриншоту
Здесь перечислены баги, найденные визуально на предоставленном скриншоте страницы поиска.
### BUG-001

| Поле | Значение |
|------|----------|
| **Приоритет** | **P0** (Критический) |
| **Заголовок** | На странице отображается сообщение "Ошибка, обновите страницу" |
| **Описание** | При открытии страницы выдачи в верхнем правом углу отображается системное сообщение об ошибке |
| **Шаги воспроизведения** | 1. Открыть главную страницу Avito<br>2. Перейти в раздел "Животные"<br>3. Выбрать категорию "Породистые кошки" -> "Персидские"  |
| **Фактический результат** | Видим текст "Ошибка, обновите страницу" |
| **Ожидаемый результат** | Страница загружена корректно, сообщения об ошибках отсутствуют |
| **Окружение** | Web-версия сайта, браузер Chrome |
### BUG-002

| Поле | Значение |
|------|----------|
| **Приоритет** | **P2** (Средний) |
| **Заголовок** | Неверный порядок категорий в хлебных крошках: "Персидская > Кошки" |
| **Описание** | В хлебных крошках нарушена логическая иерархия категорий |
| **Шаги воспроизведения** | 1. Открыть главную страницу Avito<br>2. Перейти в раздел "Животные"<br>3. Выбрать категорию "Кошки" → породу "Персидская"<br>4. Посмотреть на хлебные крошки |
| **Фактический результат** | Хлебные крошки отображаются как: "Главная > Животные > Персидская > Кошки" |
| **Ожидаемый результат** | Хлебные крошки должны отображаться как: "Главная > Животные > Кошки > Персидская" (от общего к частному) |
| **Окружение** | Web-версия сайта, браузер Chrome |
### BUG-003
| Поле | Значение |
|------|----------|
| **Приоритет** | **P2** (Средний) |
| **Заголовок** | Несоответствие количества объявлений в кнопке фильтра |
| **Описание** | Количество найденных объявлений отображается по-разному в разных элементах интерфейса |
| **Шаги воспроизведения** | 1. Открыть главную страницу Avito<br>2. Перейти в раздел "Животные"<br>3. Выбрать категорию "Кошки" → породу "Персидская"<br>4. Посмотреть на заголовок страницы<br>5. Посмотреть на синюю кнопку "Показать объявления" в нижней части панели фильтров |
| **Фактический результат** | В заголовке страницы указано число **19** ("Котята в Москве 19"), а на кнопке внизу фильтров указано число **8** ("Показать 8 объявлений") |
| **Ожидаемый результат** | Количество объявлений должно быть одинаковым во всех элементах интерфейса |
| **Окружение** | Web-версия сайта, браузер Chrome |
### BUG-004
| Поле | Значение |
|------|----------|
| **Приоритет** | **P1** (Высокий) |
| **Заголовок** | В выдаче отображается товар другой породы ("Экзотическая") при выбранном фильтре "Персидская" |
| **Описание** | Система игнорирует выбранный фильтр породы и показывает нерелевантные товары |
| **Шаги воспроизведения** | 1. Открыть категорию "Животные" -> "Кошки"<br>2. В фильтре "Порода" выбрать "Персидская"<br>3. Просмотреть карточки в выдаче |
| **Фактический результат** | В результатах присутствует карточка "Экзотический короткошёрстный котёнок" |
| **Ожидаемый результат** | В выдаче должны отображаться только объявления породы "Персидская" |
| **Окружение** | Web-версия, браузер Chrome |

### BUG-007
| Поле | Значение |
|------|----------|
| **Приоритет** | **P1** (Высокий) |
| **Заголовок** | Не работает фильтр по полу ("Мальчик"/"Девочка") |
| **Описание** | Выбор пола в фильтрах не влияет на результаты выдачи |
| **Шаги воспроизведения** | 1. В блоке "Пол" выбрать "Мальчик"<br>2. Дождаться обновления выдачи<br>3. Проверить пол животных в карточках |

| **Ожидаемый результат** | После выбора "Мальчик" должны отображаться только объявления с самцами |
| **Окружение** | Web-версия, браузер Chrome |
### BUG-008
| Поле | Значение |
|------|----------|
| **Приоритет** | **P1** (Высокий) |
| **Заголовок** | Не работает фильтр по диапазону цен |
| **Описание** | Ввод минимальной и максимальной цены не фильтрует объявления |
| **Шаги воспроизведения** | 1. В блоке "Цена" ввести диапазон (например, 1000 – 30000)<br>2. Нажать "Показать объявления"<br>3. Проверить цены в карточках |
| **Фактический результат** | В выдаче присутствуют товары с ценой выше 30 000 руб. или ниже 1 000 руб. |
| **Ожидаемый результат** | Должны отображаться только объявления в заданном ценовом диапазоне |
| **Окружение** | Web-версия, браузер Chrome |
### BUG-009
| Поле | Значение |
|------|----------|
| **Приоритет** | **P2** (Средний) |
| **Заголовок** | Некорректный фильтр "Сначала из Можайска": отображается район, не соответствующий выбранному региону Москва |
| **Описание** | Интерфейс предлагает сортировку по конкретному району (Можайск), хотя в поиске выбран весь регион "Москва" |
| **Шаги воспроизведения** | 1. Открыть сайт Avito<br>2. В блоке выбора региона убедиться, что выбрана "Москва"<br>3. Перейти в раздел "Животные"<br>4. Выбрать категорию "Породистые кошки" -> "Персидские" |
| **Фактический результат** | Отображается активный переключатель "Сначала из Можайска" |
| **Ожидаемый результат** | Переключатель должен соответствовать выбранному региону (например, "Сначала из Москвы") |
| **Окружение** | Web-версия, браузер Chrome |
### BUG-010
| Поле | Значение |
|------|----------|
| **Приоритет** | **P2** (Средний) |
| **Заголовок** | Некорректно отображаются фото товаров |
| **Описание** | Изображения обрезаются или смещаются внутри контейнера карточки |
| **Шаги воспроизведения** | 1. Открыть страницу выдачи объявлений<br>2. Визуально осмотреть сетку карточек товаров |
