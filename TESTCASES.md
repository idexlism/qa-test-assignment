# Тест-кейсы для платформы модерации объявлений
## Десктопная версия сайта

### TC-001: Проверка работы фильтра "Диапазон цен"
*   **Заголовок:** Проверка корректности фильтрации объявлений по диапазону цен
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1.  В левой панели фильтров найти блок "Диапазон цен"
    2.  В поле "От" ввести минимальное значение (например 1000)
    3.  В поле "До" ввести максимальное значение (например 5000)
    4.  Проверить цены в отображаемых карточках объявлений.
*   **Ожидаемый результат:** В выдаче отображаются только те объявления, цена которых находится в пределах от 1000 до 5000 включительно
*   **Постусловия:** Сбросить фильтры диапазона цен

### TC-002: Сортировка объявлений по возрастанию цены
*   **Заголовок:** Проверка сортировки объявлений от дешевых к дорогим
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. В блоке сортировки найти поле "Сортировать по"
    2. В выпадающем списке выбрать опцию "Цена"
    3. В соседнем поле "Порядок" выбрать опцию "По возрастанию"
    4. Проверить порядок цен в карточках объявлений.
*   **Ожидаемый результат:** Список объявлений отсортирован по возрастанию цены

### TC-004: Проверка работы фильтра "Категория"
*   **Заголовок:** Проверка фильтрации объявлений по выбранной категории
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. В левой панели фильтров найти блок "Категория"
    2. В поле выбора категории нажать на выпадающий список
    3. Выбрать любую категорию из списка (например "Животные" или "Услуги")
    4. Дождаться обновления списка объявлений
    5. Проверить, к какой категории относятся отображенные объявления
*   **Ожидаемый результат:** В выдаче отображаются объявления, принадлежащие только выбранной категории (например, "Животные"). Объявления из других категорий скрыты.
*   **Постусловия:** Нажать кнопку "Сбросить" в блоке фильтров для возврата к начальному состоянию

### TC-005: Проверка работы тогла "Только срочные"
*   **Заголовок:** Проверка отображения только срочных объявлений при активном тогле
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. Найти переключатель с подписью "Только срочные"
    2. Активировать переключатель (перевести ползунок вправо)
    3. Проверить список отображаемых карточек объявлений.
*   **Ожидаемый результат:** В списке отображаются только те объявления, которые имеют признак срочности. Объявления без этого признака скрываются.
*   **Постусловия:** Выключить переключатель "Только срочные" (вернуть в исходное положение)

### TC-006: Проверка кнопки ручного обновления "Обновить"
*   **Заголовок:** Проверка кликабельности и реакции кнопки "Обновить"
*   **Предусловия:** Открыта страница "Статистика" (/stats).
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. Нажать на кнопку "Обновить"
    2. Проверить реакцию таймера обновления
*   **Ожидаемый результат:**
    - Кнопка нажимается
    - Таймер сбрасывается

### TC-007: Проверка кнопки запуска таймера
*   **Заголовок:** Проверка переключения статуса автообновления кнопкой "Play"
*   **Предусловия:** Открыта страница «Статистика» (/stats). Статус: "Автообновление выключено"
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. Нажать на кнопку "Play"
    2. Проверить изменение текста статуса и иконки кнопки
*   **Ожидаемый результат:**
    - Появляется таймер обновления
    - Иконка кнопки меняется на "Пауза"
    - Кнопка реагирует на нажатие.
        
### TC-008: Проверка кнопки остановки таймера
*   **Заголовок:** Проверка переключения статуса автообновления кнопкой "Пауза"
*   **Предусловия:** Открыта страница "Статистика" (/stats). Таймер запущен (видна кнопка паузы)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. Нажать на кнопку "Пауза", которая появилась после запуска
    2. Проверить изменение текста статуса и иконки кнопки
*   **Ожидаемый результат:**
    - Текст статуса возвращается в состояние "Автообновление выключено"
    - Кнопка снова отображается как "Play"
    - Автоматическое обновление прекращается.

## Мобильная версия сайта

### TC-009: Переключение в тёмную тему (Светлая -> Темная)
*   **Заголовок:** Проверка переключения интерфейса на тёмную тему
*   **Предусловия:** Открыта главная страница в мобильной версии. Интерфейс находится в светлой теме
*   **Окружение:** Mobile (samsung galaxy s25), Google Chrome
*   **Шаги:**
    1. Нажать на кнопку с иконкой Луны в правом верхнем углу
    2. Проверить изменение цветовой схемы и иконки.
*   **Ожидаемый результат:**
    - Фон страницы становится тёмно-серым
    - Текст становится светлым для контраста.
    - Иконка в кнопке меняется на Солнце
*   **Постусловия:** Вернуть тему по умолчанию
