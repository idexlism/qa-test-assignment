# Тест-кейсы для платформы модерации объявлений
## Десктопная версия сайта

### TC-001: Проверка работы фильтра "Диапазон цен"
*   **Заголовок:** Проверка корректности фильтрации объявлений по диапазону цен
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1.  В левой панели фильтров найти блок "Диапазон цен"
    2.  В поле "От" ввести минимальное значение (например 1000)
    3.  В поле "До" ввести максимальное значение (например 5000)
    4.  Проверить цены в отображаемых карточках объявлений.
*   **Ожидаемый результат:** В выдаче отображаются только те объявления, цена которых находится в пределах от 1000 до 5000 включительно
*   **Постусловия:** Сбросить фильтры диапазона цен

### TC-002: Сортировка объявлений по возрастанию цены
*   **Заголовок:** Проверка сортировки объявлений от дешевых к дорогим
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. В блоке сортировки найти поле "Сортировать по"
    2. В выпадающем списке выбрать опцию "Цена"
    3. В соседнем поле "Порядок" выбрать опцию "По возрастанию"
    4. Проверить порядок цен в карточках объявлений.
*   **Ожидаемый результат:** Список объявлений отсортирован по возрастанию цены

### TC-004: Проверка работы фильтра "Категория"
*   **Заголовок:** Проверка фильтрации объявлений по выбранной категории
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1. В левой панели фильтров найти блок "Категория"
    2. В поле выбора категории нажать на выпадающий список
    3. Выбрать любую категорию из списка (например "Животные" или "Услуги")
    4. Дождаться обновления списка объявлений
    5. Проверить, к какой категории относятся отображенные объявления
*   **Ожидаемый результат:** В выдаче отображаются объявления, принадлежащие только выбранной категории (например, "Животные"). Объявления из других категорий скрыты.
*   **Постусловия:** Нажать кнопку "Сбросить" в блоке фильтров для возврата к начальному состоянию
