# Тест-кейсы для платформы модерации объявлений
## Десктопная версия сайта

### TC-001: Проверка работы фильтра "Диапазон цен"
*   **Заголовок:** Проверка корректности фильтрации объявлений по диапазону цен
*   **Предусловия:** Открыта главная страница со списком объявлений (/, /list)
*   **Окружение:** Desktop (1920x1080), Google Chrome
*   **Шаги:**
    1.  В левой панели фильтров найти блок "Цена"
    2.  Ввести минимальное значение (например 1000)
    3.  Ввести максимальное значение (например 5000)
    4.  Проверить цены в отображаемых карточках объявлений.
*   **Ожидаемый результат:** В выдаче отображаются только те объявления, цена которых находится в пределах от 1000 до 5000 включительно
*   **Постусловия:** Сбросить фильтры диапазона цен
